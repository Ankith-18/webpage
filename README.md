# 🚀 React Vite Tailwind Admin Dashboard

A modern Admin Dashboard built using **React + Vite + Tailwind CSS + Recharts**.

This project includes:

- 📊 Area Chart & Radar Chart (Recharts)
- 📁 Collapsible Sidebar
- 🔔 Notification Dropdown
- 👤 Profile Dropdown
- 📋 Clients Table
- 📦 Projects Table
- 🎨 Fully Responsive Layout
- ⚡ Fast Development with Vite

---
## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- Recharts
- React Icons
